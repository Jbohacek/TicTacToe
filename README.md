# Tic Tac Toe
#### English Language
An assignment from school, where the purpose was to create a lightweight game of variable-field sandboxes.

##### TicTacToe_1
This is the first version of TicTacToe where I managed to create a 3x3 field where you can play TicTacToe.
I also made a 3x3 bot to play against - there is an option for the player to win against the bot.
`BotGame.cs` - `HracMuzeVyhrat`
True - Can win
False - Can NOT win

The code is written in Czech, since I didn't know whether the professor, who is checking, wants Czech or English, it is clear to me that English is better.

##### TicTacToe_2
Already improved version of the project, where it is now possible to set the size of the game field when starting the application.
I've also added an "Undo button" that undoes all changes.
I had to remove the feature of playing against bots, as it would be too time consuming.

#### Czech language
Zadání ze školy, kde bylo účelem vytvořit lehkou hru na piškvorky s variabilním polem.

##### TicTacToe_1
Jedná se o první verzi piškvorek, kde se mi podařilo vytvořit 3x3 pole, kde se dají hrat piškvorky.
Udělal jsem též 3x3 **bota**, proti kterému se dá hrát - je tam možnost, aby hráč mohl vyhrát proti botovy.
`BotGame.cs` - `HracMuzeVyhrat`
	True - může vyhrát
	False - Nemůže vyhrát

Kod je psán v češtině, jelikož jsem nevěděl zda profesor, kontrolující chce češtinu či angličitnu, už je mi jasné, že je lepší angličitna.

##### TicTacToe_2
Už vylepšená verze projektu, kde už je možné nastavit velikost herního pole, při začátku aplikace.
Přidal jsem taky "Undo button", které vrátí všechny změny.
Je také možnost nastavit na kolik se hraje.
Funkci proti botovy jsem musel odstranit, jelikož by byla moc časově náročná.
